# Student Management System 🎓
A full-stack web application designed to manage student records efficiently. Built using the Django framework and styled with Bootstrap 5.

## 🚀 Features
*   **Full CRUD Functionality**: Create, Read, Update, and Delete student profiles.
*   **Dynamic Search**: Filter through student records by first or last name in real-time.
*   **Secure Access**: Integrated User Authentication to protect administrative actions (Add/Edit/Delete).
*   **Responsive Design**: A clean, mobile-friendly UI using Bootstrap cards and tables.

## 🛠️ Tech Stack
*   **Backend**: Python, Django
*   **Frontend**: HTML5, CSS3, Bootstrap 5
*   **Database**: SQLite (Development)
*   **Version Control**: Git & GitHub
