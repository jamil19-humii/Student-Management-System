# SchoolSystem: Student Analytics and Management 🎓

Hey there! This is a project I built to move beyond basic coding and into Full-Stack Development. It is a management tool that handles student data, but I also added a Phase 2 analytics layer to track academic performance and attendance trends.

## What this app does

1. Complete Lifecycle Management: I built full CRUD logic, so you can register a student, update their details if they change majors, or remove records completely.
2. Performance Analytics: Each student has a dedicated Dashboard page that calculates their average grade and attendance percentage on the fly using Django backend logic.
3. Relational Data: The app does not just store names. It links students to their specific grades and attendance records using database relationships.

## The Under the Hood Stuff

1. Framework: Python and Django, which handles all of the routing, views, and database logic.
2. UI and UX: Bootstrap 5, because I wanted it to look clean, modern, and professional on my Acer laptop.
3. Data Storage: SQLite, which is perfect for this development phase and handles the relational tables smoothly.
4. Version Control: Managed via Git and GitHub to track my progress from a basic list to a full dashboard.
